# Chrome Web Store Descriptions

## 日本語版 (Japanese)

### 短い説明 (Short Description)
日々のスケジュール管理をサイドパネルで。Googleカレンダー連携とリマインダー通知に対応した便利なタイムテーブル拡張機能。

### 詳細説明 (Detailed Description) - プレーンテキスト版

SideTimeTable で毎日のスケジュール管理を効率化！

SideTimeTableは、Chromeのサイドパネルで今日の予定を素早く確認・管理できる拡張機能です。Googleカレンダーとの完全連携で、忙しい毎日をスマートにサポートします。

【主な機能】

• サイドパネル表示: ワンクリックで今日のスケジュールをサイドパネルに表示
• Googleカレンダー連携: 複数のGoogleカレンダーを同期し、個人・チーム予定を一元管理
• カレンダーグループ: 「仕事」「プライベート」などグループを作成し、ワンクリックで表示切替
• ローカルイベント管理: Google以外の予定も直接追加・編集可能（タイムライン上のドラッグで即座に作成も可能）
• 繰り返しイベント: 毎日・毎週・毎月・平日の繰り返し予定を簡単作成
• リマインダー通知: 予定の◯分前にデスクトップ通知でお知らせ（Google予定の自動リマインドにも対応）
• メモパネル: サイドパネル下部に折りたたみ可能なメモ欄を常備
• ダークモード: ダークテーマに完全対応、テキスト色も自動調整
• カスタマイズ設定: カラーテーマ、作業時間、休憩時間などを自由に設定（色覚対応プリセット付き）
• 初回ガイド: セットアップウィザードと対話型チュートリアルですぐに使い始められる
• 多言語対応: 日本語・英語に対応

【こんな方におすすめ】

- 毎日のスケジュール確認を効率化したいビジネスパーソン
- 複数のGoogleカレンダーを使い分けている方
- 仕事とプライベートの予定を一元管理したい方
- Googleカレンダーを使わず、シンプルなデイリープランナーとして使いたい方

【使い方】

1. 拡張機能をインストール
2. Googleアカウントと連携（連携しなくてもローカル予定のみで利用可能）
3. 表示したいカレンダーを選択
4. サイドパネルで今日の予定を確認・管理

【プライバシー】

カレンダーデータは安全に処理され、必要最小限のアクセス権限（通知、アラーム、ID/カレンダー連携 など）のみを使用します。カレンダーへの書き込みは出欠回答の更新のみに使用されます。データは外部サーバーに送信されません。

## English Version

### Short Description
Daily schedule management in your side panel. Timetable extension with Google Calendar integration and reminder notifications.

### Detailed Description - Plain Text Version

Streamline Your Daily Schedule Management with SideTimeTable!

SideTimeTable is a Chrome extension that lets you quickly view and manage today's events in a convenient side panel. With full Google Calendar integration, it smartly supports your busy daily life.

[Key Features]

• Side Panel Display: View today's schedule in the side panel with one click
• Google Calendar Integration: Sync multiple Google Calendars to manage personal and team schedules in one place
• Calendar Grouping: Create groups like "Work" or "Personal" and switch views with one click
• Local Event Management: Add and edit non-Google events directly, or drag on the timeline to create instantly
• Recurring Events: Easily create daily, weekly, monthly, or weekday recurring events
• Reminder Notifications: Get desktop alerts before events (with auto-reminders for Google events)
• Memo Panel: A collapsible memo area at the bottom of the side panel for quick note-taking
• Dark Mode: Full dark theme support with automatic text color contrast
• Customizable Settings: Color themes, work hours, break times, and more (with color blindness presets)
• Getting Started Guide: Setup wizard and interactive tutorial to get you up and running quickly
• Multilingual: Available in English and Japanese

[Perfect For]

- Business professionals who want to streamline daily schedule checking
- Users managing multiple Google Calendars
- Those looking to centrally manage work and personal events
- Anyone who wants a simple daily planner without Google Calendar

[How to Use]

1. Install the extension
2. Connect with your Google account (or use it as a standalone local planner)
3. Select calendars to display
4. View and manage today's events in the side panel

[Privacy]

Your calendar data is processed securely with minimal required permissions (notifications, alarms, identity/calendar integration, etc.). Calendar write access is used only for updating RSVP responses. Data is not transmitted to external servers.
