SideTimeTable

今日の予定管理ツール

概要 機能 インストール方法 スクリーンショット 開発者情報 プライバシーポリシー

概要

SideTimeTableは、日々のスケジュールをしっかり管理するためのChrome拡張機能です。Googleカレンダーと同期し、日々の予定をサイドパネルに表示します。これにより、大事な瞬間を見逃すことなく、個人的なイベントも管理できます。

機能

  • クリック一つでサイドパネルから日々のイベントにアクセス。
  • Googleカレンダーとシームレスに同期してイベントを表示・管理。
  • ローカルイベントを簡単に追加・管理（繰り返しイベント対応）。
  • リマインダー通知機能で大事な予定を見逃さない。
  • 初回セットアップウィザードとチュートリアルで簡単スタート。
  • カスタマイズ可能な表示設定とカラースキーム。

インストール方法

  1. Chrome ウェブストアでインストール
  2. 「Chromeに追加」ボタンをクリックします。
  3. 表示されるダイアログで、拡張機能の追加を確認します。
  4. インストール後、ツールバーのアイコンをクリックしてサイドパネルを開きます。
