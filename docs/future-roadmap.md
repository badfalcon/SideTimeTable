# SideTimeTable 開発ロードマップ

> **現在のバージョン**: v1.8.0（2026年3月16日リリース）

> このロードマップは、SideTimeTable の今後の開発方針を示すものです。
> ユーザーのフィードバックや優先度の変化に応じて、内容は随時更新されます。

---

## プロダクトの核心

SideTimeTable は**カレンダーアプリの代替ではありません**。

Chrome のサイドパネルに常駐し、**作業の流れを止めずに「今」と「次」を把握する**ための時間認識ツールです。

### 大切にしていること

- **一瞬で把握** — タイムラインを一目見るだけで、今日の残り時間と次の予定がわかる
- **作業を止めない** — タブ切替もウィンドウ切替も不要。視線を横に向けるだけ
- **今日に集中** — 週や月ではなく「今日」にフォーカスし、目の前の行動を支える
- **シンプルに、でも強力に** — 最小限のUIで最大限の時間認識を提供する

今後の開発は、すべてこの哲学に沿って進めます。

## Phase 1: v1.9.x（2026年 Q2）— 「今」をもっと見やすく

サイドパネルを一瞬見ただけで、必要な情報がすべてわかる状態を目指します。

### 次の予定カード

- [ ] タイムラインの上部に「次のイベント」を常時表示するコンパクトなカード
- [ ] 開始までの残り時間をカウントダウン表示（例:「スタンドアップ まで 23分」）
- [ ] Meet リンクや場所情報へのワンタップアクセス

### 今日のサマリー

- [ ] 1日の時間配分を一行で表示（例:「会議 3h ・ 空き 4.5h ・ 残り 2h」）
- [ ] 業務時間内の空き時間を視覚的にハイライト

### 操作の効率化

- [ ] 拡張機能アイコンに当日のイベント数をバッジ表示
- [ ] イベント検索 — タイトルをキーワードでフィルタリング
- [ ] 通知サウンドオプションの追加

## Phase 2: v2.0.0（2026年 Q3）— 集中を守る

「次の予定を把握する」だけでなく、「集中できる時間を最大化する」ためのサポートに踏み込みます。

### フォーカスタイム

- [ ] 会議の合間の空き時間を自動検出し、「集中可能ブロック」として視覚表示
- [ ] ワンクリックでフォーカスタイムをローカルイベントとして登録
- [ ] 会議密度に応じた休憩リマインダー（例: 3連続会議のあとに休憩を提案）

### ミーティング準備

- [ ] イベント開始が近づくと、関連リンク（Meet / 資料URL）をカード上部に展開表示
- [ ] イベントの説明文プレビュー（ホバーまたは展開）

### キーボード操作

- [ ] `j` / `k` でイベント間を移動、`Enter` で詳細表示
- [ ] `n` で次のイベントにジャンプ
- [ ] クイック追加のキーボードショートカット

### 内部改善

- [ ] `src/services/` ディレクトリの導入 — `background.js` の責務分離
- [ ] テストスイートの導入（`time-utils.js`、`EventLayoutManager` 等）

## Phase 4: v3.0（2027年 前半）— 時間認識の深化

「今日の予定を見る」から「今日の時間をどう使ったか振り返る」へ。サイドパネルの特性を活かしながら、1日の時間体験を豊かにします。

### 1日の振り返り

- [ ] 1日の終わりに時間の使い方サマリーを表示（会議/集中/空き時間の内訳）
- [ ] 「今日の達成」をメモパネルと連携して記録
- [ ] 週次トレンドの簡易グラフ（過去7日の会議時間推移 — サイドパネルに収まるミニチャート）

### スマート通知

- [ ] 次のイベントの内容に応じた通知カスタマイズ（Meet付きなら早め通知、終日イベントは朝一回）
- [ ] 「そろそろ移動」リマインダー（場所付きイベント向け）

### イベント入力の進化

- [ ] 自然言語入力 —「明日14時 打ち合わせ」のようなテキストからイベント作成
- [ ] テンプレート機能 — よく使う予定をワンタップで登録

### 内部改善

- [ ] パフォーマンス最適化（大量イベント時のレンダリング効率化）
- [ ] プロバイダー抽象化レイヤー（将来のマルチカレンダー対応の基盤）

## 将来の検討事項

スケジュール未定ですが、プロダクトの方向性に合うアイデアです。

- **ポモドーロタイマー連携** — フォーカスタイム中のタイマー表示
- **ブラウジングコンテキスト連携** — 閲覧中のページに関連するイベントをハイライト
- **Slack / Teams ステータス同期** — カレンダーの予定に基づくステータス自動更新
- **ウィジェット表示** — サイドパネル以外のコンパクトなアクセス手段（ポップアップ等）
- **クロスデバイス同期** — ローカルイベントの端末間同期
- **マルチカレンダープロバイダー** — Outlook / CalDAV との連携

---

## やらないこと

プロダクトの核心を守るために、以下の方向には進みません。

- **週表示・月表示** — サイドパネルの幅に合わず、「今日に集中」の哲学から外れる
- **フルカレンダー機能** — Google Calendar の代替ではなく、補完であり続ける
- **複雑なスケジューリング** — Calendly のような予約・共有機能は対象外
- **詳細な時間追跡** — Toggl のようなタイムトラッキングツールとは役割が異なる
